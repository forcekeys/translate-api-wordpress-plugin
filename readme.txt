Translate your WordPress content using Forcekeys Translation API. Supports posts, pages, WooCommerce products and more.

== Description ==

Forcekeys Translation API is a powerful WordPress plugin that allows you to translate your website content using the Forcekeys Translation API.

### Features

* **Automatic Translation** - Translate posts, pages, and custom post types
* **WooCommerce Integration** - Translate product names, descriptions, prices, and more
* **Interactive Shortcodes** - Allow users to translate content on click
* **Language Selector** - Beautiful dropdown or list style language selector
* **Translation Caching** - Built-in caching to reduce API calls and improve performance
* **Batch Translation** - Translate multiple texts at once
* **Auto-detect Language** - Automatically detect source language
* **70+ Languages** - Support for over 70 languages

### Supported Languages

English, French, Spanish, German, Italian, Portuguese, Russian, Chinese, Japanese, Korean, Arabic, Dutch, Polish, Turkish, and many more.

== Installation ==

1. Upload the `forcekeys-translation` folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Translation > Settings and enter your API key (get one at https://forcekeys.com)
4. Configure your default language settings

== Frequently Asked Questions ==

= Where do I get an API key? =

Sign up at [forcekeys.com](https://forcekeys.com) and create an API key in your dashboard.

= Is there a free trial? =

Yes, we offer a free tier with limited translations. Check our pricing at https://forcekeys.com/pricing

= Does this plugin support WooCommerce? =

Yes! The plugin includes full WooCommerce integration. Enable it in the settings.

= Can I translate my entire website? =

Yes, use the language selector shortcode to allow visitors to translate the entire page content.

= How does caching work? =

The plugin stores translations in your database to avoid redundant API calls. You can configure cache duration in settings.
